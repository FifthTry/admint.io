Carousel 2

We have developed the carousel component for admint.io, which matches the design currently displayed on the live site.

How to use?

To use carousel component you need to import lib inside your .ftd file.
-- import: fifthtry.github.io/admint.io/lib

Once you import lib inside your .ftd file, use the carousel component by adding all required attributes of it.

Inside your .ftd file, -- integer $current-active: 1 variable to be added, which sends integer as index for each slide. Pass $current-active variable to each slide components reset attribute. index attribute of each slide component should be +1 (as shown in below code snippet).
Input
-- integer $current-active: 1


-- lib.carousel:
$active-item: $current-active
